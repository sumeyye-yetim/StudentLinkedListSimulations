# Student Linked List Simulations 🔗

Two data structure simulation problems implemented in Java using custom linked lists.

## Problems

### Problem 1 — Loose the Letter (Circular Linked List)
A circular one-way linked list simulates a letter-elimination game. Each round, a randomly selected player loses a letter from their name. When a player's name becomes empty, they are eliminated. The last remaining player wins.

### Problem 2 — Student Message Transfer Line (Doubly Linked List)
A custom doubly linked list simulates students passing a message along a line. The message changes direction upon reaching hub students or the end of the line. Each student tracks how many letters in the messages matched their name.

## Tech Stack
`Java` `Linked Lists` `Data Structures`

## File Structure
- `LinkedList.java` — Core linked list implementation
- `Student.java` / `StudentNode.java` — Student node structures
- `StudentList2.java` / `StudentNode2.java` — Doubly linked list implementation
- `TestClass1.java` / `TestClass2.java` — Test classes for both problems
- `names.txt` — Input file with student names
